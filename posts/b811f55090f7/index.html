CFA三级考点笔记（十二）：案例分析技巧

发表于2022-01-28|更新于2022-01-28|CFALevel3
|字数总计:1.4k|阅读时长:4分钟|阅读量:

案例分析技巧部分在考试总成绩中的占比约为未确定

第37章：案例分析技巧——机构投资者

1.流动性管理

持有非流动资产（Illiquid Asset），需要管理现金流入（Inflow）流出（Outflow）以满足需求。

  1. 流动性风险管理工具（在2020 Mock出现考题）
    1. 流动性分析和变现时间（Liquidity Profiling and Time-to-cash Tables）
      步骤：确定现金流情况；创建流动性分类时间表/流动性预算（Liquidity Budget）
    2. 流动性再平衡和承诺资本（Rebalancing and Commitment Capital）
      1. 流动性再平衡：通过交易资产，使资产再次达到战略资产配置（SAA）的水平。
        方法：系统性（Systematic），定期/百分比；自动调整（Automatic Adjustment），市场影响
      2. 承诺资本：有限合伙人LP承诺未来支付给一般合伙人GP的资金。
    3. 压力测试（Stress Testing）：考虑市场出现危机情景下的流动性，同时考虑资产/负债。
    4. 衍生品（Derivatives）：用于调整风险敞口。
  2. 非流动性风险溢价（Illiquidity Premium）：承担流动性风险取得的风险溢价。
    • Illiquidity Premium = R(Illiquidity Asset) – R(Marketable Asset)
    • Illiquid Asset Price = Marketable Asset Price – Value of Put Option

2.案例分析技巧

  1. 战略资产配置（Strategic Asset Allocation，SAA）
    承担非流动性风险可改善投资组合多样化和预期回报，并获得更广泛的投资策略机会。
  2. 流动性管理（Liquidity Management）
    流动性弱化：市场走弱；屏蔽门（Activation of Gates）；平滑效应（Smoothing）。
  3. 经理人选择（Asset Manager Selection）：需考虑道德操守和职业行为准则
  4. 战术资产配置（Tactical Asset Allocation）
    赋予经理更多权限，可围绕SAA基准在范围内临时调仓。
  5. 资产配置再平衡（Asset Allocation Rebalancing）

第38章：案例分析技巧——私人财富管理

1.初入职场阶段

  1. 明确财务目标：购置房产；组建家庭；长期财务安全；舒适的退休生活。
  2. 风险敞口识别
    1. 人力资本丰富度：行业与技能；年龄；职场经理；工作流动性。
    2. 不同工作性质的人力资本：公务员/教师类似债券；非公务员/教师类似股票。
    3. 资产与负债：资产包括存款/退休福利/人力资本；负债包括当前债务/未来支出现值。
  3. 风险敞口评估：收入风险；意外死亡风险；车祸/修理费；责任风险；购置房产。
  4. 风险管理策略
    1. 收入风险：建立至少6个月支出的储蓄。
    2. 意外死亡风险：通过人寿保险解决经济困难。
    3. 车祸和修理费：第三方强制责任险+额外储蓄。
    4. 购置房产：购房时机选择；购房后风险管理。
    5. 其他风险：财产保险；私人意料保险。
    6. 监测结果与风险敞口：定期对风险管理策略结果和风险敞口进行监测审查。

2.事业发展期

  1. 明确财务目标：最大限度增加家庭福利、减少意外；计划抚养开支；退休生活准备。
  2. 风险敞口识别与评估
    1. 收入和意外死亡：收入（政府补助/福利；残疾险）；意外死亡（金融资产增幅/人寿险）。
    2. 投资组合风险：金融资产与人力资本的关系；房地产与人力资本的关系。
    3. 退休储蓄计划：强制性社保；公司额外养老金；退休总收入计算；养老金福利变化。
    4. 其他风险：健康风险；财产风险；责任风险。
  3. 风险管理策略
    1. 人寿保险：根据人力资本计算；根据未来生活开支计算。
    2. 投资风险建议：金融资产与家庭成员职业相关性；房地产与本地行业相关性/就业情况。
    3. 退休计划建议：家庭预期退休收入缺口；增加退休储蓄以弥补缺口；增加养老金账户贡献。
    4. 其他建议：更新财产保险以反映房屋市场价值；补充私人健康险；完善个人风险策略。

3.财富快速积累期

  1. 财务目标与相关风险
    1. 财务目标：未来10-15年内持续工作；拥有舒适退休生活；为子女提供支持；留下遗产。
    2. 相关风险与管理方法：家庭拥有大量储蓄/投资；家庭成员职业为公务员/教师。
    3. 退休生活目标风险：失业；政府养老金改革；投资风险；通胀风险。
    4. 投资组合分析：子女抚养，需保证生活无法自理的子女支出；遗产取决于其他完成度。
    5. 资产配置分析：基于目标的方法（Goal-based Approach），分解为多个子投资组合。
  2. 风险管理策略
    1. 收入风险：可减少保费支出：储蓄金额足够并持续自我保险；购买了残疾/人寿保险。
    2. 退休储蓄建议：以最高限额缴纳私人养老金储蓄；采用风险更低/更平衡的资产配置。
    3. 投资组合建议：定期增加投资金额；更多配置于通胀保值债券；投资策略说明（IPS）。

4.退休前期

  1. 明确财务目标：退休生活舒适；为子女提供支持/购买房产；留下遗产；跨国养老。
  2. 退休资产与养老金提取：一次性提取养老金；年金。
  3. 退休收入建议：提供通胀保值的收入。优先提取政府养老金，其次雇主养老金。
  4. 退休前投资组合建议：年跌幅≤20%；投资于易变现产品；对冲通胀；不投资房产。
  5. 退休后投资组合建议：少于一半投资于全球股票；大于一半投资于低风险资产。
