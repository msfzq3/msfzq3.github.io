CFA三级考点笔记（六）：固定收益组合管理

发表于2022-01-13|更新于2022-04-12|CFALevel3
|字数总计:5k|阅读时长:16分钟|阅读量:

固定收益组合管理部分在考试总成绩中的占比约为15%-20%

第18章：固定收益组合管理概述

1.固定收益证券的角色

  • 分散化投资风险：固定收益证券与其他类别资产相关系数较低。
  • 经常性现金流：付息债券可以产生票息，满足现金流需求。
  • 通胀对冲：通胀挂钩债券（Inflation-linked）根据通胀率调整面值；浮动利率债。

2.投资目标

  1. 基于负债的目标/资产负债管理（Asset-liability Management，ALM）
    1. 现金流匹配（Cashflow Matching）：保证固定收益的现金流入精准覆盖未来的负债支出；
    2. 久期匹配（Duration Matching）：使资产负债的久期匹配，对冲利率变化的风险；
    3. 或有免疫（Contingent Immunization）：盈余（Surplus）超过阈值的部分允许主动管理；
    4. 时间匹配（Horizon Matching）：短期通过现金流匹配覆盖，长期通过久期匹配覆盖。
  2. 基于总体收益的目标
    1. 纯粹指数型（Pure Indexing）：尽可能的复制一个基准指数，主动收益与主动风险均为0；
    2. 增强指数型（Enhanced Indexing）：与基准保持联系，并寻求一定的主动收益；
    3. 主动管理型（Active Management）：允许投资组合与基准之间存在较大风险因子差异。

CFA三级主要涉及的久期概念包括：麦考利久期（Macaulay Duration）、修正久期（Modified Duration）、有效久期（Effective Duration）
麦考利久期（Macaulay）：属于时间概念，单位是年，用于衡量投资债券取得所有现金流的平均时间。
修正久期（Modified）：属于利率敏感度的概念，没有单位，用于衡量利率变动1单位时，债券价格变动的幅度。
有效久期（Effective）：本质和修正久期一样，但也适用于表示含权债券的利率敏感度。若债券不含权，其有效久期和修正久期相等。
此外，在修正久期的基础上，也引入了衡量债券价格变动的具体数额的相关概念，包括：美元久期（Money Duration）、BPV/PVBP
美元久期Money Duration = 修正久期Modified Duration * 市场价值Market Value
BPV（PVBP） = 修正久期Modified Duration * 市场价值Market Value * 基点0.0001

3.收益分析

需要记忆公式，可能会单独考每一项的计算

  • E(R) = Yield Income + Rolldown Return + E(Price Changed) + E(Currency Gains) - E(Credit Loss)
    • 滚动收益率Rolling Yield = Yield Income + Rolldown Return
    • 现金收益率Yield Income = Annual coupon payment / Current bond price
    • 收敛收益率Rolldown Return = (End Price – Begin Price) / Begin Price
    • 预期利率导致的价格变动率E(Price Changed) = -MD*ΔYield + 0.5*Convexity*ΔYield^2
    • 预期信用损失E(Credit Loss) = Probability of Default*Loss Given Default(Expected Loss Severity)
    • 预期汇率利得E(Currency Gains)

4.其他因素

  1. 流动性：固定收益市场流动性低于股票，影响交易速度与市场冲击程度
    发行主体（Issuer）；信用质量（Credit Quality）；发行量（Issue Size）；发行时间；期限（Maturity）
    影响：低流动性的矩阵定价（Matrix Pricing）；组合构建的买卖价差太大；使用固收衍生品
  2. 杠杆（Leverage）：放大收益同时放大亏损
    杠杆组合收益Rp = （整体收益-资金成本）/自有资金 = (Re*(Ve+Vb)-Rb*Vb)/Ve
    增加杠杆：期货、互换、结构化金融工具、回购协议（Repos）、证券出借
  3. 税收（Tax）：资本利得的税率一般低于利息收益的税率
    如需出售持仓的债券，需综合考虑债券的当前损益、税率、免税账户等因素执行卖出决策
    分离式管理账户（Separately Managed Account）/集合投资工具（Pooled Investment Vehicle）

第19章：被动型投资策略

1.负债驱动型策略（Liability-driven Strategy）

  1. 资产负债管理（Asset-Liability Management，ALM）：同时考虑资产端与负债端的管理策略。
    负债类型：第一类金额与时间都确定；第二类金额确定；第三类时间确定；第四类都不确定。
    1. 资产驱动型负债（Asset-driven Liabilities，ADL）：给定资产组合，调整负债的结构，该策略较为罕见。
    2. 负债驱动型投资（Liability-driven Investing，LDI）：分析未来负债发生的时点和金额，以此指导资产管理。
  2. 单一负债管理策略（Single Liability Managing Strategy）
    此处的单一负债（Single Liability）指的是需要在指定期限偿付的某项负债，可以看作零息债券。
    1. 现金流匹配：
      • 零息债券面值 = 单一负债金额；
      • 零息债券到期时间 = 单一负债支付时间。
    2. 久期匹配（重点）：单一负债实现久期匹配需满足以下3个条件：
      1. 债券资产初始市值 ≥ 单一负债现值；
      2. 债券资产的麦考利久期 ≈ 单一负债的投资期限（对于单一负债来说，久期=投资期限）；
        • 久期缺口Duration gap = 麦考利久期Macaulay duration – 投资期限Investment horizon
      3. 债券资产的凸度（Convexity） ≈ 单一负债凸度。
        由于单一负债 ≈ 零息债券，零息债券现金流集中，具有最小凸度，因此此处也意味着令凸度最小化。
        • 凸度衡量利率变动对久期的影响，Convexity = (MD^2 + MD + Dispersion)/(1 + Cash flow yield)^2
    3. 利率免疫（Interest Rate Immunization）：假设债券无违约风险，现金流匹配优于久期匹配。
    4. 结构化风险（Structural Risk）：收益率曲线旋转、移动使免疫策略失效的风险。
      降低结构化风险的方法：
      1. 最好的方法是使用现金流匹配，可以完全避免结构化风险；
      2. 若无法实现现金流匹配，则应降低债券组合的凸度，尽量减少现金流的分散程度。

      降低债券组合的凸度，意味着使现金流相对集中在久期附近，那么当收益率曲线旋转时，对债券价值的影响更小
      但是，如果收益率曲线是平行移动、方向未知的，提高凸度反而能够减缓收益率波动带来的影响

  3. 多重负债管理策略（Multiple Liabilities Managing Strategy）
    1. 主要方法：
      1. 现金流匹配
      2. 久期匹配（重点）：多重负债实现久期匹配需满足以下3个条件：
        1. 债券资产现值 ≥ 多重负债现值；
        2. 债券资产的修正久期/基点价值 ≈ 多重负债的修正久期/基点价值；
          • 基点价值（Basis Point Value，BPV）：表示到期收益率变动1个基点时，债券价格的变动幅度。

          单一负债使用麦考利久期Macaulay Duration；
          多重负债使用修正久期Modified Duration或基点价值BPV；
          含权负债使用有效久期Effective Duration。

        3. 债券资产的凸度（Convexity） ≥ 多重负债凸度。
    2. 次要方法：衍生品叠加（广泛用于收益确定型养老金Defined Benefit）
      1. 国债期货（10-year T-note Futures Contract）：每张100000USD
        • 交割金额 = FP * CF + AI
        • 合约数量N = (资产BPV - 负债BPV) / 期货BPV
      2. 利率互换（Interest Swap）：
        • 收固定付浮动利率互换MD = 固定利率债MD - 浮动利率债MD
        • 名义本金NP = (资产BPV - 负债BPV) / 互换BPV * 100
      3. 互换期权（Swaption）：receiver收固定付浮动，payer收浮动付固定，期权价值最低为0
      4. 互换期权衣领组合（Swaption Collar）：买入1个收款/付款，同时卖出1个付款/收款
    3. 或有免疫（Contingent Immunization）：主动与被动相结合的策略，盈余购买高风险产品
  4. 相关风险
    • 模型风险（Model Risk）：计量错误、假设错误（比如收益率曲线移动的假设）、基准不合理
    • 信用风险（Credit Risk）：主要指交易对手方违约的风险（Counterparty Risk）
    • 流动性风险（Liquidity Risk）：主动投资需要关注，对采用或有免疫的投资组合尤其重要

2.被动投资（Passive Investment）

  1. 被动管理 VS 主动管理
    • 被动管理（Passive Management）：与基准保持一致，衡量跟踪误差（Tracking Error）。
    • 主动管理（Active Management）：对市场利率变化进行分析及预测，追求超额收益，允许较大偏离。
  2. 纯粹指数策略（Pure Indexing）：即完全复制法（Full Replication）
    纯粹指数策略的特征/难点：
    • 固收市场更庞大广泛，完全复制的交易成本较高；
    • 大多数债券流动性偏低，场外市场为主，常用矩阵定价（Matrix Pricing），与实际值有差异；
    • 指数构成可能发生变动，需要及时进行再平衡（Rebalancing）；
    • 容易受信用评级和含权债券的影响。
  3. 增强指数策略（Enhanced Indexing）
    分层抽样复制法（Stratified Sampling）/模块法（Cell Approach）：按指数特征分层，分别匹配并选择个券。
    现金流现值分布法（PV of Distribution of Cash flows）：将现金流按离散区间划分，折现加总，计算对久期的贡献。
    增强指数策略的特征/难点
  4. 另类被动投资法（Alternative Passive Investment）
    • 开放式共同基金（Mutual Funds）：按照每份额净资产价值（NAV per share）进行申赎，具备规模效应，流动性更好，分散化效果更强。
    • 交易所交易基金（ETF）：使用债券实物进行申赎，也可以在二级市场买卖，流动性强，可能存在价差。
    • 总收益互换（Total Return Swap，TRS）：收取方获得资产收益，支付方获得约定现金流，初始投资少，但存在信用风险。
  5. 环境、社会和治理（ESG）：考虑社会责任投资（Socially Responsible Investing，SRI），不适用于完全复制法。

3.基准选择

  1. 选择标准

    • 规则清晰透明；
    • 可投资性（Investability）；
    • 每日估值可知；
    • 历史调仓可知。

  2. 基准偏离的原因：随着到期日临近，久期不断下降；质量低劣问题（Bums Problem）；市场偏好发生变化。

    Bums Problem指的是用证券自身的市值做权重，导致单个证券对Portfolio影响超出正常水平的问题
    在固收中，bums problem就是用国债市值加权，导致债务越高的国家在portfolio中比例越高，显然不合适，可改用国家GDP做权重
    在权益中，bums problem就是用股票市值加权，导致市值越大的股票在portfolio中占比越高，即Large Cap Bias，可改用个股基本面指标做权重

  3. Smart Beta：将短期与长期投资分开处理；使用系统性、规则化的量化方法。

4.债券组合（Bond Portfolio）

现金流分布 适用场景
梯形组合（Laddered） 平均投资到各个期限 收益率曲线稳定并上倾，可以有效降低利率风险
杠铃型组合（Barbell） 集中在时间轴两端，配置超短期和超长期债券 收益率曲线变得平缓（Flattening），短端利率下降，长端利率上升。由于长端久期大，价值上升更多，可产生额外收益
子弹型组合（Bullet） 集中在久期附近，分散程度较低 收益率曲线旋转但方向不定，子弹型组合可有效降低凸度，减轻结构化风险（Structural Risk）

第20章：收益率曲线策略

1.收益率曲线稳定

  1. 买入并持有（Buy and hold）：构建与基准偏离的组合，并在一定期间保持稳定。
  2. 骑乘收益率（Riding the yield curve/Roll down）：收益率上倾，买入长期债中途卖掉。
    骑乘效应：在债券持有期，剩余期限缩短导致收益率也下滑，因此债券价格上升。
  3. 做空凸性（Sell Convexity）：凸度用于抵御利率波动；若收益率曲线平稳，可出售凸度增厚收益。
    做空凸性的方法如下：
    • 卖出已持有债券的看涨期权，或者卖出期望持有债券的看跌期权；
    • 买入可赎回债券（Buy Callable Bond），该赎回权属于卖方权利，相当于买方做空凸度；
    • 买入抵押支持证券（Buy Mortagage-based Securities），MBS发行方拥有提前还款的权利，相当于买方做空凸度；
    • 卖出可售回债券（Sell Puttable Bond），该售回权属于买方权利，相当于卖方做空凸度；
    • 集中现金流，降低现金流的分散程度。
  4. 套息交易（Carry Trade）：以低利率借款，投资到高利率的债券中，以此赚取利差。

2.收益率曲线变化

  1. 预期：平行移动
    1. 方向已知：久期管理，若未来利率下跌/上升，投资经理应该增加/缩短组合的久期。
      久期管理的方法：通过杠杆投资调整资产负债的久期；使用衍生品，如利率期货、利率互换。
    2. 方向未知：做多凸性，高凸性债券在利率上下行情况下表现都更好，更能抵御波动。
      做多凸性的方法（与做空凸性的方法相反）：
      • 买入期权（Buy Options）；
      • 卖出可赎回债券（Sell Callable Bond）；
      • 卖出抵押支持证券（Sell Mortagage-based Securities）：MBS可以视为Callable Bond；
      • 买入可售回债券（Buy Puttable Bond）；
      • 分散现金流。
  2. 预期：非平行移动
    1. 蝶式组合（Butterfly）：做多杠铃（两端），同时做空子弹（中间），使组合久期中性。
      蝶式价差Butterfly Spread = 2 * Medium Yield - Short-term Yield – Long-term Yield
    2. 鹰式组合（Condor）：做多翅膀（近端+远端），同时做空身体（较近端+较远端），使组合久期中性。

    蝶式和鹰式本质上类似，只是蝶式组合会做多两份中期债券，而鹰式组合做多一份偏近期和一份偏远期的债券
    也就是说，鹰式组合的“身体部分”比蝶式组合更长

  3. 预期：曲率增加

曲率增加（More Curvature）表示短期和长期利率下降，中期利率上升，曲线呈驼峰状。
- 此时，中期债券价格下跌，短期和长期债券价格上升。
- 因此应做空中间，做多两端，可选择做空子弹、做多杠铃、做空蝶式/鹰式组合。

3.套息交易（Carry Trade）

  1. 市场内套息（Intra-market）：只使用同一种货币，同一条收益率曲线。
    关注期限错配（Maturity Mismatch），用短期低息融资支持长期高息投资，即短融长投。
    • 在回购市场融资并投资于债券（Buy a bond and finance it in the repo market）；
    • 持有收固定付浮动利率互换合约（Enter a receive fixed and pay floating interest rate swap）；
    • 做多债券期货合约（Take a long position in a bond future）。
  2. 市场间套息（Inter-market）：使用多种货币进行套息。
    1. 存在外汇风险敞口：利用无抛补利率平价关系的不成立（Uncovered Interest Arbitrage）进行套利。
      • 借入低利率货币，投资于高利率货币；
      • 利用外汇互换，收取高利率货币，支付低利率货币；
      • 通过外汇远期合约（FX Forward），将负债转换为低利率货币。
    2. 不存在外汇风险敞口：对收益率曲线更陡峭的市场，采取短融长投；对收益率曲线更平坦的市场，采取短投长融。
      • 配对交易（Pair Trading），判断两国市场各自长短期利率的差异，从利差小的国家融资，向利差大的国家投资。

现实中，由于资本无法完全自由流动，且利率汇率并非完全遵循平价公式，受各种政策和社会因素影响，套息交易往往难以实现。

4.收益率曲线交易评估（Evaluate）

E(R) = Yield Income + Rolldown Return + E(ΔPrice from Yield) – E(Credit Loss) + E(Currency Gain)

第21章：信用风险管理策略

1.信用风险（Credit Risk）

  1. 风险识别

    1. 利率风险：
      • 经验久期（Empirical Duration）：通过历史实证数据推算的久期；有效久期（Effective Duration）：通过理论公式推导的久期
      • 无风险利率通常与信用利差负相关：当利率上升/下降时，往往意味着经济过热/衰退，信用风险下降/上升，导致信用利差下降/上升
      • 由于利率与信用利差存在负相关关系，利率变动对价格的实际影响低于理论值，经验久期 < 有效久期（Effective Duration）
      • 垃圾债券/高收益债券（High-Yield Bond）受信用利差影响更大，对利率变动的敏感度更小，垃圾债经验久期 < 国债经验久期
    2. 信用风险：违约概率（Probability of Default，PD）；违约损失率（Loss Given Default，LGD）
    3. 流动性与交易：高收益/非投资级债券流动性低、价差更大，常用分层抽样复制法投资

  2. 信用利差（Credit Spread）

    1. 信用利差的类别

      1. 基准利差（Benchmark Spread）：信用债券与无风险债券之间的收益率差异

      2. G-Spread：信用债券与政府债券（G-Bond）之间的利差，不同期限可用插值法处理

      3. I-Spread：信用债券与标准互换利率（Standard Swap Rate）之间的利差

      4. 零波动利差Z-Spread：又称静态利差（Static Spread），用不同期限利率作为贴现率倒推

      5. 期权调整利差OAS（Option-Adjusted Spread）：在Z-Spread的基础上剔除期权价值的利差

        可赎回债券（Callable Bond）：OAS < Z-Spread；
        可售回债券（Puttable Bond）：OAS > Z-Spread

    2. 计算信用超额收益（Excess Return to Credit Risk）（记忆）
      持有期超额收益 = 期初利差 * 持有时间 - 由于利差变化造成的市场价格损失

      • 不考虑违约可能性：超额收益XR = 期初利差s * 持有期t - 持有期利差变动Δs * 利差久期SD
      • 考虑违约可能性：期望超额收益EXR = 期初利差s * 持有期t - 持有期利差变动Δs * 利差久期SD - 年化违约概率p * 违约损失率L * 持有期t

      注意：当债券的利差Spread降低，持有该债券的超额收益XR上升，该债券的价格也上升。此处Spread本质上和利率类似，与债券价格负相关。

2.信用策略（Credit Strategy）

  1. 自下而上的策略（Bottom-up Approach）
    相对价值分析（Relative Value Analysis）：在特征相似的债券中，选择相对价值较高者
    具体操作顺序：板块选择；个券选择；利差曲线
    其他考量因素：债券结构；发行日期；市场供给；发行规模
    组合构建：被动构建（Passive Construction）；主动构建（Active Construction）
    证券选择：替代品选择（次优）；指数化；持有现金
  2. 自上而下的策略（Top-down Approach）
    五要素：宏观因素；概略的板块分类；信用质量；信用周期预期；信用利差变化预期
    信用质量分析：平均信用评级；平均OAS；平均利差久期；久期与利差乘积（DTS）
    具体操作方法：行业配置；利率敞口；国家和外汇敞口；利差曲线
  3. 两种策略的优缺点
    • 自上而下：基于宏观因子获取信用收益；
    • 自下而上：在个券上具有信息优势。
  4. 环境、社会、公司治理（ESG）
    相对价值考量；准则约束；风险计量；影响力投资

3.信用市场风险

  1. 流动性风险：二级市场流动性；产业机构变化
    管理方法：持有现金；管理头寸规模；持有高流动性非基准债券；信用违约互换；ETF
  2. 尾部风险：情景分析（Scenario Analysis）
    管理方法：投资组合分散化；尾部风险对冲
  3. 国际信用组合：新兴市场；流动性、外汇及法律风险
  4. 结构化金融工具：收益高于传统；分层投资；相对价值盈利；增加组合分散化程度
    • 住房抵押贷款支持证券（MBS）：流动性更优；房地产敞口；利率波动性敞口；周期投资
    • 资产支持证券（ABS）：抵押品类型众多（汽车贷款、租赁、信用卡、学生个人贷款）
    • 担保债务凭证（CDOs）：相对价值套利优势；违约相关系数敞口；附杠杆投资收益
    • 资产担保债券：由信贷机构（商业银行）发行，由抵押贷款/公共债务担保，收益风险偏低
