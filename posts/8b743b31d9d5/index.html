CFA三级考点笔记（七）：权益组合管理

发表于2022-01-13|更新于2023-02-20|CFALevel3
|字数总计:3.8k|阅读时长:12分钟|阅读量:

权益组合管理部分在考试总成绩中的占比约为10%-15%

第22章：权益类资产投资组合管理概述

1.权益类资产的角色

主要作用：资产增值（Capital Appreciation）；获取分红收益；风险分散；对冲通胀风险

2.权益类资产分类

  1. 按照规模与风格矩阵分类
    优点：便于管理；覆盖面广泛；提供了比较基准；可用于判断公司发展阶段
    缺点：分类标准不固定且不统一；随着时间改变分类结果可能也会变化
    • 根据市值规模（Size）：大盘股（Large Cap）；中盘股（Mid Cap）；小盘股（Small Cap）
    • 根据风格（Style）：价值股（Value）；成长股（Growth）；混合股（Blend/Core）
  2. 按照地缘政治分类：发达市场（Developed）；新兴市场（Emerging）；前沿市场（Frontier）
  3. 按照经济活动分类：生产导向（Production-oriented）；市场导向（Market-oriented）
  4. 按照指数基准分类

3.权益类投资收入与成本

收入：分红收入（Dividend）；融券（Lending）；辅助投资策略（Ancillary Investment Strategy）
成本：管理费用；业绩费用；行政费用；营销与分销费用；交易费用
掠夺性交易（Predatory Trading）：买入纳入指数的股票，卖出剔出指数的股票，拉高成本

4.以股东身份参与管理

投资者可以以股东身份在一定程度上参与公司管理，这是权益类投资的最大特点。
搭便车效应（Free Ride）：小股东可以搭大股东的便车，从而参与公司管理的监督、投票

5.主动管理与被动管理

考量：取得超额收益的信息/能力；客户偏好；挂钩指数；投资要求；主动的风险；税收

第23章：被动权益资产投资

1.选取基准指数

  1. 必要条件：严格的编制规则（Rule-based）；透明（Transparent）；可投资（Investible）

  2. 成分股选择：

    1. 穷尽法（Exhaustive）：穷尽符合指数要求的所有股票，如上证综指；
    2. 选择法（Selective）：选择符合条件/排名靠前的部分股票，如上证50。

  3. 指数构造方法：

    1. 市值加权（Market Cap/Value-weighted）：个股权重=个股总市值占比，低误差/少调仓，要求充足流通市值（Free-float）；
    2. 等权重（Equally-weighted）：个股权重相等，每只股票持仓市值相等，避免集中于大盘股，但需要频繁再平衡/调仓，且容量受限；
    3. 价格加权（Price-weighted）：个股权重=个股价格占比，每只股票持仓数量相等，易受分红配股影响，相对使用较少；
    4. 基本面加权（Fundamental-weighted）：根据基本面指标加权，长期价值视角，使用较少。
    5. 有效成分股
      • 指数集中度（Concentration Level）：使用Herfindahl-Hirschman指标，HHI = ∑wi^2
      • 指数中有效成分股的数量（Effective Number of Stocks）：N = 1 / HHI

    指数成分股的调整方式：缓冲法Buffering；重构法Reconstituting；分拆法Packeting
    缓冲法Buffering提供“缓冲区”，当成分股不符合指数要求时，不会立刻调出指数，减少频繁交易降低成本。
    重构法Reconstituting在成分股不符合指数要求时立刻调出，并重新按要求构建指数，保证准确性。
    分拆法Packeting将成分股仓位分拆成几部分，在成分股不符合指数要求时仅调出部分仓位，若现象持续再全部调出。

  4. 因子投资策略（Factor-based）：又称Smart Beta，根据股票因子来配置指数
    常见因子如Fama-French五因子模型：贝塔因子、价值因子、规模因子、动量因子、盈利因子。

    1. 收益导向（Return-oriented）：分红、动量、各类基本面加权；
    2. 风险导向（Risk-based）：波动率加权、指数波动率最优化；
    3. 分散化导向（Diversification-oriented）：等权重指数、最大分散化。

    管理费用：指数投资最低；被动因子投资中等；主动投资最高。

2.被动股票投资管理方法

  1. 集合投资工具（Pooled Investments）
    • 共同基金（Mutual Funds）：交易费用较低，投资更专业；但可能存在现金滞留（Cash Drag），追加投资未能及时购买股票。
    • 交易所基金（ETFs）：交易便利，支持做空、杠杆和税收递延；但存在买卖溢价和额外手续费，可能出现流动性短缺。
  2. 衍生品法（Derivatives-based Approaches）：使用衍生品合成标的指数。
    一般不会长期持续使用，可用于短期调整覆盖头寸（Overlay）。
  3. 基于指数的独立管理账户（Separately Managed Equity Index-based Portfolio）

3.投资组合的构建方法

  1. 完全复制法（Full Replication）：完全复制指数成分股，最小化追踪误差，易于理解。
    要求资产规模足够大，且标的指数成分股完全具备可交易性，整体交易成本较高。
  2. 分层抽样法（Stratified Sampling）：基于Cell Matching，先将成分股分类，再分别抽取。
    分类必须互斥（Mutually Exclusive）且穷尽（Exhaustive），解释相关性不强的收益因子。
  3. 最优化法（Optimization）：在约束条件下，最大化/最小化投资组合的某个特征。
    又称量化技术法（Quantitative & Technical），考虑相关系数，但依赖历史数据且操作复杂。
  4. 混合法（Blended Approach）
    对大盘股/高流动股使用完全复制法；对中小盘股/低流动股使用分层抽样法或最优化法。

4.追踪误差管理

超额收益率：Excess Return = 组合收益率Return of Portfolio-基准收益率Return of Benchmark
追踪误差：Tracking Error = (Variance(Rp-Rb))^0.5
产生原因：交易费用；持股占指数成份股的比例；日内交易价格；佣金费用；现金滞留

5.被动投资的收益与风险

归因分析（Attribution Analysis）：用于事后（Ex-post）分析收益来源
融券（Securities Lending）：信用风险（Credit）；抵押品风险（Collateral）；流动性风险
参与公司管理：被动投资必须持有多只成分股，可能导致对单只个股的关注和了解不足

第24章：主动权益资产投资

1.主动投资管理

  1. 基本面分析法（Fundamental Approach）：主观判断，个人投研能力与经验，聚焦个股分析
  2. 量化分析法（Quantitative Approach）：客观统计模型，基于大量历史数据，归纳最优化

2.主动投资策略

  1. 自下而上的策略（Bottom-up Strategies）
    从个股出发，再形成行业市场观点，最终构建组合。
    1. 基本面分析：基于个体公司的商业模式、品牌、相对优势、公司管理进行分析。
    2. 价值股投资：关注“安全边际”，即公司股票的内在价值和实际股价的差距。
      • 相对价值（Relative Value）：横比同行业股票的各类价值指标（市盈率、市净率等），选择高于平均水平的个股。
      • 逆向投资（Contrarian Investing）：与市场共识相悖，投资于低收益/股息的周期股，待公司盈利好转反弹。
      • 高质量价值（High-quality Value）：除了关注估值，更重视财务实力、经营管理、持续盈利能力等企业质量指标。
      • 红利投资（Income Investing）：选择股息收益率较高、股息增长率为正的股票，核心目的是赚取股利收益。
      • 深度价值（Deep Value）：关注低估值的股票，往往是陷入财务困境、市场关注度较低的股票，可能存在市场失灵。
      • 重组和不良投资（Reconstruct and Distressed Investing）：投资于不良债务，可能涉及资本结构的改变。
      • 特殊情景投资（Special Situations）：关注公司的重大事件，寻找市场反应不及时和错误定价的机会。
    3. 成长股投资：寻找发展速度高于行业平均/市场平均的企业，关注营收、利润或现金流等成长质量指标。
      • GARP（Growth at Reasonable Price）：关注成长的同时，也寻求合理价格，常用PEG指标（PE除以利润增长率）。
  2. 自上而下的策略（Top-down Strategies）
    先寻求宏观环境较好的国家，再选取符合发展趋势/有政策支持的行业，最后精选个股。
    • 板块与行业轮动（Sector and Industry Rotation）
    • 波动率策略（Volatility-Based Strategies）
    • 热点主题投资（Thematic Investing）：聚焦于投资特定主题或领域的企业。
    • 影响力投资（Impact Investing）：强调实现社会环境影响力，同时追求财务回报。
  3. 基于因子投资的策略（Factor-based Strategies）
    • 风格因子（Style Factors）：价值、动量、成长、质量
    • 有回馈因子/无回馈因子（Rewarded/Unrewarded Factor）：因子是否长期与收益率显著相关
    • 对冲投资组合法（Hedge Portfolio Approach）：股票按因子分组，定期观察判断因子有效性
      缺陷：中间组未考虑；假定线性关系；可能过于集中；做空交易受限；无法剔除其他因子
  4. 积极参与型主动投资策略（Activist Strategies）
    干涉公司的经营战略、日常运营等事务，从而提升公司的内在价值，获取股票投资收益
  5. 其他策略：统计套利（Statistical Arbitrage）；事件驱动策略（Event-driven Strategies）

3.构建主动投资策略

  1. 基本面分析策略
    • 操作步骤：确定投资范围；深入筛选公司；分析、预测并估值；构建投资组合；再平衡
    • 行为偏差：确认偏差/掌控幻觉/易得性偏差/损失厌恶/过度自信/后悔厌恶
    • 其他缺陷：价值股陷阱（低PE≠高价值）；成长股陷阱（增长未达预期）
  2. 量化分析策略
    • 操作步骤：确定投资机会；获取/处理数据；回测/计算因子IC值/建模；策略评估；风控/交易
    • 行为偏差：幸存者偏差/前视偏差/数据挖掘/过拟合（Overfitting）
    • 其他缺陷：高换手导致交易费用较高；量化拥堵（Quant Crowding）

    前视偏差（Look-ahead Bias）：使用未来数据指导当期投资，如年报公告期晚于审计期
    量化拥堵（Quant Crowding）：策略因子若使用人数过多，超额收益率可能迅速下降消失

4.权益投资风格分类

  1. 基于持仓（Holding-based）：价值型、混合型、成长型；大盘、中盘、小盘。构建9类矩阵。
  2. 基于收益（Return-based）：将收益率对各类风格指数线性回归，根据回归系数敞口判断。

前者更准确，但需要知道持仓、判断标准多样；后者使用简便，但数据问题会降低准确性。

第25章：主动管理投资组合构建

1.主动管理收益来源与构成

收益构成：因子权重（Factor Weightings）；阿尔法技巧（Alpha Skills）；头寸控制（Sizing Positions）

  • 超额收益率RA = Σ(βpk - βbk) * Fk + (α + ε)
  • 超额收益率RA = 战略配置收益Σ（Rewarded Factors） + 战术配置收益α + 运气收益ε
  • 期望超额收益E(RA) = IC * (BR) ^ 0.5 * TC * σ = 信息系数 * 决策次数 ^ 0.5 * 转移系数 * 超额波动率
  • 其中，IC表示预测与真实收益的相关性，BR表示独立决策次数，TC表示观点转为实操的能力。

2.构建主动投资组合（Portfolio Construction）

  1. 投资组合的方法选择（比较辨析）
    1. 系统性管理（Systematic）VS酌情判断（Discretionary）
      • 系统性管理：证券选择更分散（Diverse selection of Securities），通常使用量化模型进行决策。
      • 酌情判断：证券选择范围更小，持仓可能更集中（Concentrated），依赖投资经理的独立判断。
    2. 自上而下（Top-down）VS自下而上（Bottom-up）
      • 自上而下：强调宏观因素（Macro Factors），当环境不好时持有现金；
      • 自下而上：强调个股特征（Firm-specific Characteristics），倾向于因子识别与择时。
  2. 目标与约束条件
    1. 主动投资比例：Active Share = 0.5*Σ|Weight in Portfolio – Weight in Benchmark|
      若Active Share = 0.6，即表示有60%的持仓仓位和基准仓位存在偏离。
      Active Share越高，表示投资经理的主动管理程度越深，一般来说收费标准也会更高。
    2. 主动风险：Active Risk = [Σ(Active Return)^2/(T-1)]^0.5，即主动超额收益的标准差。
      主动超额收益：Active Return = Portfolio Return - Benchmark Return
      主动风险来源：因子暴露程度与基准的差异；个股的异质性风险。

    注意：Active Share和Active Risk并非正相关！
    Active Share仅代表投资组合与基准的持仓比例差异，而Active Risk还需要考虑行业/个股间的相关性和异质性。
    比如“超配金融股，低配科技股”和“超配科技股，低配金融股”具有相同的Active Share，但其因子暴露明显有差异，Active Risk不同。

    根据Active Share和Active Risk，可判断投资风格：行业轮动、个股选择或追踪指数。
    行业轮动（Sector Rotation）：通常会集中持仓于特定行业，具有较高的Active Risk，可能高也可能低的Active Share（视分散化程度）。
    个股选择（Stock Picking）：相比于基准，对特定个股的仓位会增加，具有较高的Active Share，可能高也可能低的Active Risk（视分散化程度）。
    追踪指数（Closet Indexing）：尽可能与基准保持一致，具有较低的Active Share和Active Risk。

    1. 约束条件
      绝对收益框架：最大化夏普比率-设置仓位上限、波动率上限、市场规模上下限
      相对收益框架：最大化信息比率-设置偏离基准上限、追踪误差上限、市场规模上下限

3.风险预算分配（Risk Budget）

  1. 绝对风险（掌握计算：资产/因子对组合风险的贡献率）
    • 投资组合收益率方差Var(Portfolio) = Σ[Wi * Σ(Wj * Cij)]
    • 每个资产对总方差的贡献率CVi = Wi * Σ(Wj * Cij) = Wi * Cip
    • 如果新增股票与组合的协方差高于原有股票，则组合的总体风险也随之上升
  2. 相对风险
    • 投资组合超额收益的方差AVar(Portfolio) = Σ{(Wi-Bi)*Σ[(Wj-Bj)*RCij]}
    • 每个资产对总方差的贡献率CAVi = (Wi-Bi)*Σ[(Wj-Bi)*RCij] = (Wi-Bi)*RCip

4.风险监控

启发性约束（Heuristic Constraints）：主观判断和过往经验（合理但非最优化）
正式约束（Formal Constraints）：统计指标，如波动率、VaR，通常包含了对风险的预测

5.隐性成本（Implicit Costs）

市场冲击成本（Market Impact）：资产管理规模（AUM）；投资组合换手率；突发事件
滑点成本（Slippage）：股票最终成交价与挂单时均价的差额，市场波动率越大，滑点越高

6.股票策略分类

  • 纯多头策略（Long-only）：只使用多头仓位，优势是容量较大，可以承接大规模资金，管理费用相对较低；缺陷是收益来源有限。
  • 多空策略（Long Short）：可以充分表达看空的观点；充分利用杠杆和分散化投资；更好地控制敞口。
  • 多头延伸策略（Long Extension）：用多头与空头持仓合成100%仓位的策略（若简写为120/20，即多头120%+空头20%）；可以从多空两端获益，但误判风险也更大。
  • 市场中性策略（Market Neutral）：完全对冲市场风险，Beta=0；配对交易（Pair Trading）；统计套利（Statistical Arbitrage）。
