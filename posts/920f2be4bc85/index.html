CFA三级考点笔记（五）：衍生品和外汇管理

发表于2022-01-12|更新于2023-02-20|CFALevel3
|字数总计:4.2k|阅读时长:14分钟|阅读量:

衍生品和外汇管理部分在考试总成绩中的占比约为5%-10%

第15章：期权策略

1.衍生品概述（Derivatives）

衍生品：远期（Forward）、期货（Futures）、互换（Swap）、期权（Option）
期货Total Return = Price Return现货收益 + Roll Return期现价差 + Collateral Return抵押收益

  1. 衍生品分类

    • 远期承诺（Forward Commitment）：收益与标的资产价格呈线性关系，如远期、期货、互换
    • 或有索取权（Contingent Claim）：收益与标的资产价格呈非线性关系，如期权
    • 场内交易所（Exchange-traded Markets）：标准化、流动性高、监管严格、需求不够细化
    • 场外市场（Over-the-counter，OTC）：个性化、自由灵活、违约风险、监管不足、低流动性

    基差（Basis） = 现货价格Spot Price - 远期价格Forward Price
    若基差为正，说明现货价格高于远期，远期价格预期将会上升，此时应做空现货，做多远期
    若基差为负，说明现货价格低于远期，远期价格预期将会下降，此时应做多现货，做空远期

  2. 期权（Option）

    • Delta：衡量标的价格变化对期权价格的影响。Call的Delta为正数；Put的Delta为负数。
    • Gamma：衡量标的价格变化对期权Delta的影响。无论Call/Put，买入期权的Gamma为正数（Delta变化速率递增），卖出期权的Gamma为负数。
    • Vega：衡量标的波动率变化对期权价格的影响。无论Call/Put，买入期权的Vega为正数（波动率越大，期权越值钱），卖出期权的Vega为负数。
    • Theta：衡量到期时间变化对期权价格的影响。无论Call/Put，买入期权的Theta为负数（时间价值逐步衰减），卖出期权的Theta为正数。
    • Rho：衡量无风险利率变化对期权价格的影响。

    关于Delta（Δ）：看涨期权Δ∈(0,1)；看跌期权Δ∈(-1,0)；平值看涨/看跌Δ≈0.5/-0.5

2.合成衍生品策略

  • 合成衍生品头寸（Synthetic Position）或头寸等价物（Position Equivalencies）来对冲风险。
  • 期权平价公式（Put-Call Parity，PCP）：S0+P0 = C0+K*(1+r)^(-t) = C0+K*e^(-rt)
  • 备兑策略（Covered Call）：股票多头+看涨期权空头。提高收益；平稳减仓；目标止盈。
  • 保险策略（Protective Put）：股票多头+看跌期权多头。对冲下跌风险。

3.价差和期权组合策略

价差策略（Spread）：多期权组合，货币价差（Money Spread）/日历价差（Calendar Spread）

  1. 货币价差-牛市价差（Bull Spread）：买低卖高
    • 牛市看涨价差（Bull Call Spread）：买入低行权价看涨+卖出高行权价看涨
    • 牛市看跌价差（Bull Put Spread）：买入低行权价看跌+卖出高行权价看跌
  2. 货币价差-熊市价差（Bear Spread）：买高卖低
    • 熊市看涨价差（Bear Call Spread）：买入高行权价看涨+卖出低行权价看涨
    • 熊市看跌价差（Bear Put Spread）：买入高行权价看跌+卖出低行权价看跌
  3. 日历价差：买长卖短，短期空头+长期多头。预测市场短期保持平稳，长期大幅波动
  4. 跨式组合（Straddle）：期限和行权价相等的看涨期权多头+看跌期权多头
  5. 领口组合（Collar）：标的多头+看跌多头+看涨空头（S+P-C）（形状与牛市看涨价差相同）

4.隐含波动率（Implied Volatility）

  1. 隐含波动率：将期权的市场价格代入BSM模型中，反推标的资产收益率的波动率
  2. 波动率倾斜/波动率微笑
    • 波动率倾斜（Volatility Skew）：低行权价期权的隐含波动率>高行权价期权的隐含波动率。
    • 波动率微笑（Volatility Smile）：虚值期权的隐含波动率>平值期权的隐含波动率

    可以按X轴行权价格，Y轴隐含波动率的方式绘制波动率倾斜/微笑的图像
    波动率倾斜：向右下单调递减的曲线，低行权价期权的波动率更高，高行权价期权的波动率更低
    波动率微笑：凹型曲线，低行权价期权（虚值看跌）和高行权价期权（虚值看涨）的波动率都更高，中等行权价的期权波动率更低

    在出现波动率倾斜（Skew）时，可以使用风险逆转（Risk Reversal）策略
    风险逆转（Risk Reversal）：卖出波动率高估的OTM看跌期权，买入波动率低估的OTM看涨期权。

5.期权的应用

市场预期 隐含波动率上升 隐含波动率不变 隐含波动率下降
牛市（Bullish） 买入看涨 买看涨/卖看跌 卖出看跌
市场中性（Neutral） 买入跨式 日历价差 卖出跨式
熊市（Bearish） 买入看跌 买看跌/卖看涨 卖出看涨

第16章：互换、远期和期货策略

1.组合风险和收益调整

  1. 利率风险管理
    1. 利率互换（Interest Rate Swap，IRS）
      特点：不交换名义本金，只交换利息部分；结算净额交割；浮动利率取决于期初利率。
    2. 普通利率互换（Vanilla）：收/付固定利率方，相当于买/卖固定利率债，卖/买浮动利率债。
      收固定付浮动(Receive Fixed/Pay Float)可以提高久期；付固定收浮动(Pay Fixed/Receive Float)可以降低久期。
      例：互换收固定利率，到期期限5年；付浮动利率，付息间隔为6个月。该互换久期D = 5*0.75 - 0.5*0.5 = 3.5
      • 浮动利率债的久期 ≈ 付息间隔 / 2 = Interval * 0.5
      • 固定利率债的久期 ≈ 到期期限 * 75% = Period * 0.75
      • 利率互换久期（Swap Duration） = 接收类型债券久期 - 支付类型债券久期
    3. 使用互换调整久期
      互换名义本金N = 组合市值MV *（目标久期MDt - 组合久期MDp） / 互换久期MDs

      浮动方具有低MV，但同时承受高现金流风险；
      固定方锁定现金流，但MV易受市场影响。

    4. 远期利率协议（Forward Rate Agreement，FRA）
      FRA以利率本身作为标的。场外产品，优点是可以定制，但存在交易对手风险。
    5. 利率期货（掌握计算）：欧洲美元期货（Eurodollar Futures）/固收期货（Fixed-income Futures）
      1. 欧洲美元期货： 100万美元面值，90天期限；利率每下降1bp，合约价值上升25美元。
      2. 固定收益期货：标的资产为多种可交割债券，空头方可以选择交割哪一种债券。
        • 最便宜可交割券（Cheapest-to-deliver，CTD）交割价格 = 期货市场价 * 转换因子 * 合约规模
        • 最便宜可交割券美元久期BPVctd = 单份合约规模Size * （市场价格Pctd / 债券面值100） * 修正久期MDctd * 基点0.01%
        • 国债期货美元久期BPVf = 最便宜可交割券美元久期BPVctd / 转换因子CF
        • 国债期货对冲份数N = 国债期货对冲比率BPVHR = （目标BPVt - 组合BPVp） / 国债期货美元久期BPVf

        BPV即基点价值（Basis Point Value），又称美元久期，表示利率每变动一个基点0.01%引起的该资产价值的变动
        通过计算目标BPV与组合BPV的差值，再除以衍生品BPV，可求出对冲比率BPVHR，即需要多少份衍生品能够实现久期风险对冲

  2. 汇率风险管理
    1. 货币互换（Currency Swap）：交换货币现金流，名义本金不一定交换，净额交割
      • 交叉货币基点互换（Cross-Currency Basis Swap）：期初及期末各交换一次本金，按浮动利息交换，采用基点报价
      • 基点报价（Basis Rate）：交换浮动利息时，预设一个基准利率，按照浮动利率与基准的点差作为应付现金流
    2. 货币远期/期货（Currency Forward/Futures）：多头未来支付外币，空头未来收到外币
      若期现价格变动幅度相同，则为有效对冲（Effective Hedge），否则有基差风险（Basis Risk）
  3. 股权风险管理
    1. 股票互换/权益互换（Equity Swap）：无须实际买卖股票，调整股票市场的风险敞口
      • 总收益互换（Total Return Swap，TRS）：交换持有期损益和分红，但不交换所有权和投票权，存在交易对手风险
    2. 股票期货（Equity Futures）：
      目标βt*S = 组合βs*S + 合约βf * 合约数量N * 合约价值F
      1. 股指期货（考查对冲份数的计算）
        • 对冲份数N = （组合市值 / 每份期货价值） * （目标β - 组合β） / 期货β
        • 对冲后有效β（Effective Beta） = 新资产组合的价值变动Δ / 市场的价值变动Δ
      2. 现金证券化（Cash Securitization/Overlay）：现金beta为0，可以购买股票期货进行调整
  4. 波动率管理
    1. 波动率期货（Volatility Futures）：以各类波动率指数为标的，如VIX、VSTOXX、VDAX-NEW

      注意：波动率指数如VIX不能直接进行交易，只能通过期货实现做多做空。

    2. 方差互换（Variance Swap）：用资产的隐含波动率和实际波动率进行互换，到期时交换一次
      • 互换波动率本金（Vega Notional）：表示波动率每变化1%，方差互换的价值变动幅度。
      • 互换名义本金：Variance Notional = 波动率本金Vega Notional / （2 * 约定波动率Strike Volatility）
      • 结算金额：Settlement Amount = 名义本金Variance Notional *（实际方差Realized Variance - 约定方差Strike Variance）
      • 结算金额：Settlement Amount = Variance Notional * (σ^2 - Strike^2) = [Vega Notional / (2 * Strike)] * (σ^2 - Strike^2)
      • t时刻方差互换合约价值：Vt = Variance Notional * PVt(T) * (t/T*σt^2 + (T-t)/T*σT^2 - Strike^2)
        • 其中，PVt(T)表示从t时刻到T时刻的折现率：PVt(T) = 1 / (1 + Rf*(T-t)/T)

      在方差互换的计算中，实际波动率σ和约定波动率Strike都直接取数字，百分号视为单位。

2.资产配置

  1. 通过交易期货/远期/互换，改变组合的BPV（美元久期）、Beta或久期。
    • 买入/卖出债券期货可增加/减少组合的BPV
    • 买入/卖出股指期货可增加/减少组合的Beta

3.市场预期推断

  1. 美国联邦基金利率（Effective Federal Funds Rate，FFE）：最主要的隔夜拆借利率
    • 预期联邦基金利率（Expected FFE Rate） = 100 - 联邦基金期货价格（Fed Funds Futures）
      例：若题目给定，在下一次FOMC会议后到期的联邦基金期货价格为97.2，则预期FFE Rate为100-97.2 = 2.8%
    • 利率调整的概率 = （基于期货推算的FFE - 当期FFE）/（预测调整后的FFE - 当期FFE）
      例：若题目又给定，当期FFE Rate的目标范围是2.5%-2.75%。由于预期FFE Rate为2.8%，高于当前范围，即市场预期联邦基金利率将会上调
      当期FFE Rate取2.5%-2.75%的均值=2.625%；预测利率上调25bps，则调整后区间2.75%-3.00%，调整后均值=2.875%
      那么，可计算出利率上调概率（Probability of Rate Hike）：P = （2.8%-2.625%）/（2.875%-2.625%） = 70%

第17章：外汇风险管理

1.外汇风险管理概述

  1. 外汇交易（Foreign Exchange，FX）：分为即期交易、远期交易、外汇互换、外汇期权
    • 外汇现货（Spot）：交易双方于成交两个交易日内交割，报价 = 计价货币/基础货币（P/B）
    • 外汇远期（Forward）：远期交易，采用点数报价（Point），报价 = 远期与即期汇率的点差

      外汇远期的价格一般以Bid/Ask的形式，加上对应期限的点数给出。
      比如，外汇现货价格为1.52/1.55，对应一个月的点差为60/80，点差规模（Scale）为10000
      那么，买入一个月外汇远期的价格：Forward Price = Ask Price + Points / Scale = 1.55 + 80 / 10000 = 1.558

    • 外汇互换（FX Swap）：同时进行即期和远期交易，锁定未来汇率，可用于滚仓（Roll）
      • 对比交叉货币互换（Cross-Currency Swap）：都会交换本金，但外汇互换不交换利息，期限较短，采用实时汇率而不是锁定汇率。
    • 外汇期权（Currency Option）：以汇率为标的资产的期权
  2. 外汇组合收益/风险
    • 收益拆分：本币回报Rdc = （1 + 外币回报Rfc）*（1 + 汇率回报Rfx）-1
    • 风险/波动率拆分：本币σ^2 = 外币σ^2 + 汇率σ^2 + 2*外币σ*汇率σ*相关系数ρ

2.外汇风险管理决策

  1. 投资策略决策（Investment Policy）
    1. 被动对冲（Passive Hedging）：对冲掉所有的外汇风险敞口，需要定期再平衡，汇率风险最小。
    2. 选择性对冲（Discretionary Hedging）：在对冲汇率风险的基础上，将敞口上下浮动一定比例以获取额外收益。
    3. 主动外汇管理（Active Currency Management）：承担一定的汇率风险，通过主动管理获取收益。
    4. 外汇管理外包（Currency Overlay）：通过外包管理获取汇率Alpha收益，当汇率与组合相关性低时最有效。

    短期来看，风险对冲有助于平衡外汇风险；外汇市场并非完全有效，可以进行投机或套利交易。
    长期来看，汇率具有均值回归的特性，风险对冲作用不大，可能产生额外的管理费用以及抬高交易成本。
    当期限较短、风险厌恶程度高、有短期资金需求、固收资产较多、对冲成本低、市场动荡时，应选择对冲外汇风险。

  2. 主动外汇管理的基础（Base of Active Currency Management）
    1. 经济基本面分析（Economic Fundamentals）：汇率围绕长期均衡价值（Long-run Equilibrium Value）波动，获取波动收敛（Convergence）的收益。
    2. 技术分析（Technical Analysis）：历史数据包含了预测未来价格所需的信息，历史数据可能决定未来走势，历史走势可能重现。
    3. 套利交易（Carry Trade）：当无抛补利率平价不成立时，借入低利息/溢价的货币，投资于高利息/折价的货币。
    4. 波动率交易（Volatility Trade）：利用衍生品，买卖跨式/宽跨式组合（Straddle/Strangle）。
  3. 交易策略
    1. 外汇远期：非标准合约，可满足定制化需求；适用范围更广；无需缴纳保证金（Up-front Margin）；交易量大时流动性好
      滚动收益率（Roll Yield/Roll Return）：表示远期和现货价差带来的收益，Roll Yield = -Side * (远期价格F - 现货价格S) / 现货价格S
      • 当远期价格大于现货价格（升水），多头的滚动收益率为负，空头的滚动收益率为正，更倾向于空头对冲。
      • 当远期价格小于现货价格（贴水），多头的滚动收益率为正，空头的滚动收益率为负，更倾向于多头对冲。
    2. 外汇期权：保险策略；双限期权（Collar）；看跌价差；海鸥策略（Seagull）
    3. 奇异期权（Exotic Option）：敲入（Knock-in），敲出（Knock-out），二元期权（Binary）
  4. 货币对冲
    • 静态对冲（Static Hedging）：构建完成后长时间内不会改变，优点是易于实施和监控且成本更低，但对冲可能逐渐失效。
    • 动态对冲（Dynamic Hedging）：需要定期进行再平衡，灵活性较高，对冲效果更好，但交易成本更高且操作更加复杂。
    • 交叉对冲（Cross Hedge）：又称宏观对冲（Macro Hedge），管理多种外币资产时，若外币间相关性较高，且持仓分别为多头和空头，则可以相互对冲。
      • 替代对冲（Proxy Hedge）表示使用本币对冲外币风险；交叉对冲（Cross Hedge）表示使用一种外币对冲另一种外币的风险。
      • 最小方差对冲比率（Minimum Variance Hedge Ratio）：外币资产历史回报用本币计量的回归曲线斜率（Slope Coefficient）。

        计算公式：最小方差对冲比率MVHR = β = 相关系数ρ * 本币回报波动率σ(RDC) / 对冲工具波动率σ(RFX)
        若求得外币对本币的MVHR=0.8，即意味着可以使用100m的对冲工具（即Forward）对冲80m的本币。

    • 基差风险（Basis Risk）：对冲工具的价格变动与底层资产的价格变动可能不一致，导致风险暴露。
  5. 新兴市场外汇管理
    1. 新兴市场（Emerging Market）具有以下特征：
      • 市场交易量小，买卖价差大（Wider bid-ask spread），交易成本更高；
      • 由于货币市场不活跃，可能需要进行交叉货币兑换（Cross-currency），使成本增加；
      • 新兴市场货币收益呈现负偏度（Negatively Skewed），左侧肥尾，发生极端事件的风险更高。
    2. 无本金交割外汇远期（Non-deliverable Forwards，NDF）
      • 到期时根据约定汇率与实际汇率的差额，执行现金交割，而非实物交割；
      • 结算使用自由兑换的货币（如美元），无需持有远期标的货币，适用于受到外汇管制的新兴国家货币。
